# Denoiser

Training a Convolutional Neural Network for denoising path traced images

## Method 1

**Dimensions**

240p (426x240) 

**Input (240p)**

1. Noisy Path-Traced Render
2. Normals
3. Depth
4. Albedo
5. Shape Classifer
6. Emission Classifer
7. Material Classifer

**Output (240p)**

Denoised Image
